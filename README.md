# 손글씨 숫자 분류기

손으로 쓴 숫자(0~9) 이미지를 KNN으로 분류하는 머신러닝 모델입니다.

## 사용 기술
- Python
- scikit-learn (KNN, 교차 검증, 혼동 행렬)

## 작동 방식

1. **데이터 로드** — sklearn에서 손글씨 숫자 데이터를 가져옵니다.
2. **분리** — train/test로 나눕니다 (test 비율 0.2).
3. **학습** — 학습 데이터로 KNN을 학습시킵니다.
4. **평가** — 테스트 데이터로 정확도를 측정하고, 교차 검증으로
   안정적인 성능을 확인합니다.
5. **분석** — 혼동 행렬로 어떤 숫자를 어떤 숫자와 헷갈리는지 확인합니다.

## 실행 방법
1. `pip install scikit-learn`
2. `python digits.py`   (실제 파일명에 맞게)

## 배운 점

- **KNN 작동 방식**: 데이터를 저장만 해두고(게으른 학습), 예측할 때
  새 데이터와 모든 학습 데이터의 유클리드 거리를 계산해 가장 가까운
  이웃들을 찾아 다수결로 정한다.
- **하이퍼파라미터 선택**: 정확도는 n_neighbors=7이 높았지만 한 번의
  분할 운일 수 있어, 교차 검증 평균이 높고 안정적인 3을 선택했다.
- **모든 개선이 통하는 건 아니다**: weights='distance'를 시도했으나
  이 데이터에서는 효과가 없어 uniform으로 되돌렸다.
- **혼동 행렬 분석**: 모델이 9를 3, 4, 5와 주로 헷갈린다는 것을
  확인했다. 실수가 대부분 모양이 비슷한 숫자끼리 일어났다.
